# 功能

- **KernelSU**：KernelSU 是 Android GKI 设备的 root 解决方案，它在内核模式下工作，并直接在内核空间中向用户空间应用程序授予 root 权限。
- **SUSFS**：KernelSU 的附加 root 隐藏内核补丁和用户空间模块。
